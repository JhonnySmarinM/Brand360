## ğŸ“– User guide

### Step 1: Idea configuration

1. Enter your business/project description (minimum 5 characters)
2. Add keywords/values by pressing Enter (maximum 5)
3. Click on \"Generate identity\"

### Step 2: Identity selection

1. Review the proposals generated by the AI
2. Switch between plans (Free, Basic, Premium, Pro) to see different options
3. Select your favorite identity by clicking on a card
4. Click on \"Finish branding\"

### Step 3: Launch

1. View your consolidated identity
2. Check the SEO Score and generated assets
3. Download the full Brand Kit (future feature)
4. Restart the process if you want to create another brand

   ```env
   VITE_AI_API_KEY=your_api_key_here
   ```

## ğŸ› Troubleshooting

### The app is not generating identities

2. If you use `mock` mode, it should work without API keys
